# \*\*Prerequisites:\*\* 

# 

# 1\. Clone the repository using the project's Git URL 

# 2\. Navigate to the project directory

# 3\. Install dependencies: `npm install`

# 4\. Create an `.env.local` file and set the right environment variables

# 

# ```

# VITE\_Vercel\_APP\_ID=your\_app\_id

# VITE\_Vercel\_APP\_BASE\_URL=your\_backend\_url

# 

# e.g.

# VITE\_Vercel\_APP\_ID=cbef744a8545c389ef439ea6

# VITE\_Vercel\_APP\_BASE\_URL=https://my-to-do-list-81bfaad7.Vercel.app

# ```

# 

# Run the app: `npm run dev`
